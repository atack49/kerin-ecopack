# KERIN ECOPACK - Landing Page Sostenible

Landing Page One-Page diseñada para la captación de leads y promoción de empaques ecológicos. Enfocada 100% en conversión con una estética premium y responsiva.

## 🚀 Características
- **Diseño Premium:** Estética moderna basada en sostenibilidad (Dark Forest Green & Kraft).
- **100% Responsiva:** Optimizada para dispositivos móviles, tablets y escritorio.
- **Arquitectura de Conversión:** Estructura lineal diseñada para guiar al usuario hacia el formulario de cotización.
- **Integraciones Listas:** Preparado para conectar con MySQL, SendGrid (Email) y HubSpot (CRM).

## 🛠️ Stack Tecnológico
- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (Vanilla).
- **Backend:** PHP para el procesamiento de leads e integraciones.
- **Diseño:** Google Fonts (Outfit), FontAwesome 6.

## 📂 Estructura del Proyecto
```
/
├── assets/
│   └── img/         # Activos visuales de alta calidad
├── index.html       # Estructura principal
├── style.css        # Diseño y responsividad
├── app.js           # Lógica frontend y animaciones
└── submit.php       # Integración backend y APIs
```

## ⚙️ Configuración
Para activar las funcionalidades de backend, edita el archivo `submit.php` con tus credenciales de:
1. Base de Datos MySQL.
2. API Key de SendGrid.
3. Access Token de HubSpot.
